# 📌 Shjjop-Cart

Projeto de um site de e-commerce com um component de carrinho de compras funcional

---

## 🚀 Tecnologias Utilizadas

- [Linguagem/Ferramenta 1] – JavaScript
- [Linguagem/Ferramenta 2] – React
- [Linguagem/Ferramenta 3] – Tailwind CSS
- [Outras] – Context API, React-Router-Dom, React-Icons

---

## 🎯 Funcionalidades

- ✅ Funcionalidade 1 – Ex: Mostra os Cards dos produtos, estes são gerados por uma mock API
- ✅ Funcionalidade 2 – Ex: Permite adicionar itens ao Carrinho de Compras e mostra os itens que estão nesse carrinho num dropdown menu
- ✅ Funcionalidade 3 – Pagina do Carrinho mostra todos os itens, permite alterar quantos items o usuario quer levar e calcula o valor total da compra
- 🔄 Em desenvolvimento: Responsividade da pagina
- ❌ Futuro: Pagina de Pagamento, cadastro de usuarios
